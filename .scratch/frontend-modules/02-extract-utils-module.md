## Agent Brief

**Category:** enhancement
**Summary:** Consolidate drag & drop utility functions and MoveTaskArgs into a dedicated utils module, removing the stale drag_drop.rs file.

**Current behavior:**
Drag & drop logic (`on_drag_start`, `on_drop`, `move_task_with_data`) and the `MoveTaskArgs` struct live in the main app component. An unused `drag_drop.rs` file exists with similar but divergent code. There is no centralized place for cross-cutting utility logic.

**Desired behavior:**
All drag & drop utility functions and the `MoveTaskArgs` struct reside in a single `utils.rs` module. The main app component imports drag & drop utilities from this module. The stale `drag_drop.rs` file is removed. The `DraggableTask` and `DropZone` component wrappers are available for import alongside the utility functions.

**Key interfaces:**
- `on_drag_start(task_id: String, column_id: String, order: u32, ev: DragEvent)` — sets `"text/plain"` data transfer with pipe-delimited format
- `on_drop(target_column_id: String, new_order: u32, ev: DragEvent, on_refresh: impl FnOnce())` — parses drag data, calls `invoke("move_task", ...)`, triggers refresh
- `move_task_with_data(target_column_id: String, new_order: u32, data: &str, on_refresh: Rc<dyn Fn()>)` — same logic but with pre-extracted data string
- `MoveTaskArgs { id: String, columnId: String, order: u32 }` — serde struct for IPC, note `columnId` field casing
- `DraggableTask` and `DropZone` — Leptos components that wrap the utility functions

**Acceptance criteria:**
- [ ] All drag & drop functions and `MoveTaskArgs` live in `src/utils.rs`
- [ ] The main app component imports drag & drop utilities from the new module
- [ ] The old `src/drag_drop.rs` file is removed
- [ ] Unit tests verify drag & drop utility functions with mock data
- [ ] Drag & drop functionality continues to work correctly in the app

**Out of scope:**
- Changing drag & drop behavior or the pipe-delimited data format
- Backend IPC command changes
- Moving ColumnView or TaskCard components
- Adding new utility functions unrelated to drag & drop
